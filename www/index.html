NetResponse: probabilistic algorithms for functional network analysis
Leo Lahti*¹, Olli-Pekka Huovilainen, and António Gusmão

(1) Department of Information and Computer Science, Aalto University, Finland
* Corresponding author; (contact information)

The NetResponse algorithm provides model-based tools for functional network analysis, and a global map of network activation patterns. The method utilizes functional measurements of the network states across multiple conditions to detect functionally coherent local regions in the network. The algorithm also characterizes the condition-specific responses of such local subnetworks.

The R/BioC package contains also an implementation of variational Dirichlet process Gaussian mixture model (Kurihara et al., 2007).

Your feedback and contributions are welcome. See the project page at BioConductor, or contact project admin.

Software & Documentation

The applicability of the models has been demonstrated by case studies in computational biology, to investigate transcriptional activity in genome-scale pathway networks across normal human body.
  • R/BioConductor: For installation instructions, see the project page at BioConductor. For further documentation, see the package vignette.

  • Visualization: Preliminary tools are available to visualize netresponse output. Install the netresponse visualization package and check the package vignette (in package source, inst/doc/) for examples.
  • Matlab: Source code for Matlab is also available.

Background

Techniques for model-based network analysis are implemented in R, an open source environment for statistical computing.

The implementations are based on probabilistic models and variational learning. The probabilistic framework deals rigorously with the uncertainties associated with small sample sizes, and allows incorporation of prior information in the analysis.
